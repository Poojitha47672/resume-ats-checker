# Resume ATS Checker

An AI-powered tool that analyzes your resume against a job description and gives you an ATS compatibility score, matched/missing keywords, strengths, and improvement suggestions.

## Live Demo
🔗 [Coming soon — Vercel link]

## Tech Stack

**Backend:** Python, FastAPI, Google Gemini AI  
**Frontend:** React + Vite, Tailwind CSS, Axios  
**Deployed on:** Render (backend) + Vercel (frontend)

## Features

- ATS compatibility score (0–100)
- Matched and missing keywords
- Resume strengths analysis
- Actionable improvement suggestions
- Powered by Gemini 2.5 Flash

## Run Locally

### Backend
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
